# Black Pink Phone

Sitio web de tienda de telefonos inspirado en estilo e-commerce premium (black + pink), con catalogo visual, filtros por marca, buscador, carrito simulado y secciones comerciales.

## Archivos principales

- `index.html`: estructura completa de la pagina.
- `styles.css`: estilos visuales y diseno responsive.
- `script.js`: interacciones (filtros, busqueda, carrito, menu movil).

## Como ejecutar

Desde la carpeta del proyecto:

```bash
python3 -m http.server 8000
```

Luego abre:

`http://localhost:8000`
